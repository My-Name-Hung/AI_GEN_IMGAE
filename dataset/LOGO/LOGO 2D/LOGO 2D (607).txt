Mô tả Hình ảnh:
Logo hiển thị tên thương hiệu "BÌNH YÊN" được viết bằng phong cách chữ viết tay nghệ thuật màu xanh lá cây đậm. Bên dưới có dòng chữ "TIỆM CÀ PHÊ" được sắp xếp với khoảng cách đều đặn bằng font chữ in hoa đơn giản cùng màu.

Yếu tố Thiết kế:
- Hình dạng: Chữ viết tay với các nét cong mềm mại và tự nhiên
- Kiểu chữ: Chữ viết tay nghệ thuật cho tên chính, font sans-serif đơn giản cho slogan
- Bảng màu: Xanh lá cây đậm (teal) trên nền kem nhạt
- Nền: Màu kem/be nhạt, đơn giản
- Phong cách: Thủ công, tự nhiên, hiện đại tối giản
- Tông điệu: Yên bình, thư thái, ấm áp

---

Visual Description:
The logo displays the brand name "BÌNH YÊN" written in an artistic handwritten style in dark teal color. Below it, the text "TIỆM CÀ PHÊ" is evenly spaced in simple uppercase letters using the same color.

Design Elements:
- Shape: Handwritten script with soft, flowing curves and natural strokes
- Typography: Artistic handwritten script for main name, simple sans-serif for tagline
- Color Palette: Dark teal on light cream background
- Background: Light cream/beige, clean and simple
- Style: Handcrafted, organic, modern minimalist
- Mood: Peaceful, relaxed, warm
